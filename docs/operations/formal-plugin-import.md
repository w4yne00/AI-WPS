# 正式插件手工导入说明

正式插件手工导入包来源于：

- `formal-plugin-kit/`
- `packaging/build_formal_plugin_kit.sh`

生成命令：

```bash
bash packaging/build_formal_plugin_kit.sh
```

生成产物：

- `dist-formal-plugin-kit/wps-ai-assistant-kit-YYYYMMDD.tar.gz`

导入时请保持插件目录名：

- `wps-ai-assistant_1.0.0`

这份包是按你现场验证通过的 `jsaddons` 宿主模式组织的，不是开放平台最小目录结构。
