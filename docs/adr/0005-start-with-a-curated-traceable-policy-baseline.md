# 首批预置规范采用小规模可追溯基线

---
status: accepted
---

首批写作规范预计控制在约 80 至 120 条，以实际去重和评审结果为准，包括 G 企技术写作基础、技术文件文体、党政公文文体和网络安全术语。规则优先来自 `yangqi-tech-writing` 的组织语体、保护边界和场景规则，以及现行公文制度和网络安全标准；每条记录规范来源及版本。国家标准内容只形成原创简要适用说明并记录标准编号，不批量复制标准正文；`yangqi-tech-writing` 派生内容保留 MIT 许可声明。该选择优先降低误报、规则冲突和现场校订成本，后续通过版本化规范包逐步扩展。
