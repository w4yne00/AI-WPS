# WPS AI Task Pane Blue-Gray Theme Design

日期：2026-05-27
选定方向：`A 雾蓝银白`

## 目标

将任务窗口和 Ribbon 图标从现有深绿色强调风格调整为明亮、克制的蓝灰白配色，使长时间文档办公场景中的阅读与操作反馈更清爽。

## 设计范围

- 任务窗口只调整颜色令牌及由这些颜色驱动的视觉状态：页面背景、玻璃面板、主按钮、输入焦点、Markdown 引用块、表头和链接。
- Ribbon 图标保持现有按钮语义、文件名、映射和 32 x 32 尺寸，只更新图形使用的主色/辅助色为蓝灰体系。
- 状态提示中的成功、警告、错误颜色保留语义区分，不用主题蓝覆盖状态含义。

## 配色系统

| 角色 | 颜色 | 用途 |
| --- | --- | --- |
| 背景 | `#f4f7fc` | 页面银白底色 |
| 柔光面板 | `rgba(238, 244, 251, 0.78)` | 范围条、浅表面 |
| 主文字 | `#182334` | 标题与正文 |
| 次文字 | `#647286` | 辅助说明 |
| 主强调色 | `#386ea8` | 主按钮、链接、提示标签 |
| 按压强调色 | `#2c5a8b` | 交互按压层次 |
| 强调浅底 | `rgba(56, 110, 168, 0.08)` | Markdown 表头/引用块 |

## 不变边界

- 不改 `taskpane.html` 的界面结构、控件或文本。
- 不改 `taskpane.js` 的 Markdown 渲染、任务执行、文档应用逻辑。
- 不改 `ribbon.js` 的模式路由、任务窗口打开流程或图标路径映射。
- 不改 adapter、Dify 请求数据、API URL、API Key 或工作流配置。

## 缓存与交付

当前分支中的 `v0.12.2-alpha` 尚包含待一并验证的前端/功能变更。本次主题资源并入该待交付版本，保留已存在的缓存参数与 manifest 版本，避免制造与功能无关的二次版本漂移。

## 验收

1. `taskpane.css` 中不再出现旧主题绿色强调值，主题视觉使用雾蓝银白令牌。
2. 所有 Ribbon 图标仍保持原路径、按钮对应关系和 32 x 32 PNG 规格。
3. 现有前端 smoke/helper 测试通过。
4. 对任务窗口进行可视化检查，确认正文 Markdown 的标题、段落、列表、引用块和表格仍清晰显示。
