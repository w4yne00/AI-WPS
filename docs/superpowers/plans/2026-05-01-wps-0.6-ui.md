> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Architecture:** Keep the current WPS native JS add-in and local adapter service. Use `ribbon.xml` for five entry points and a single `taskpane.html` that switches modes through the `mode` query parameter.

**Tech Stack:** WPS JS add-in, CustomUI ribbon XML, vanilla HTML/CSS/JavaScript, local Python adapter.
