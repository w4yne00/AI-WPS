# 正式插件目标机验证清单

## A. 导入与入口

- [ ] 目录名为 `wps-ai-assistant_1.0.0`
- [ ] 成功导入插件
- [ ] WPS Ribbon 中能看到 `WPS AI 助理`
- [ ] 点击后能正常拉起任务面板

## B. 适配层连通

- [ ] 点击 `刷新配置` 后状态变为可用
- [ ] 本地适配层地址确认为 `127.0.0.1:18100`
- [ ] 若不通，面板显示明确错误

## C. Word 能力

- [ ] `格式校对` 返回问题清单
- [ ] `排版预览` 返回变更计划
- [ ] `改写/续写` 返回改写结果
- [ ] `应用预览` 可执行排版写回
- [ ] `应用预览` 可执行改写写回

## D. 运行时探针

- [ ] `运行探针` 能返回 `WPS global`
- [ ] `Active document` 为 `true`
- [ ] 在有选区时 `Selection available` 为 `true`
- [ ] `Paragraph count` 大于 `0`
