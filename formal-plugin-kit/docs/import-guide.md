# 导入说明

## 目标

将正式一期插件手工导入麒麟 V10 的 WPS `jsaddons` 环境。

## 关键约束

- 插件目录名必须保持为 `wps-ai-assistant_1.0.0`
- 该包按本地 `jsaddons` 宿主模式组织，不需要 `npm` 或 `vite`
- 本地适配层默认监听 `127.0.0.1:18100`

## 导入步骤

1. 将压缩包复制到目标机并解压。
2. 手工导入目录 `wps-ai-assistant_1.0.0/`。
3. 打开 WPS，点击 `WPS AI 助理 -> 打开 AI 助理`。
4. 先点击 `刷新配置`。
5. 如果健康状态正常，再测试：
   - `格式校对`
   - `排版预览`
   - `改写/续写`
   - `运行探针`
6. 在 `排版预览` 或 `改写/续写` 后，点击 `应用预览` 验证写回行为。
